```markdown
📂 World Backup Script
A simple batch script to keep your Minecraft world safe by creating compressed backups.

✨ Features
Auto-Naming: Each backup is named with the exact date and time.

No Software Needed: Uses built-in Windows PowerShell to compress files (no WinRAR required).

Organized: Automatically creates a Backups folder to store your data.

🚀 How to Use
Place backup.bat in your main server folder.

Run the script manually whenever you want to save a copy of your world.

Your backups will appear in a folder named Backups.
```
